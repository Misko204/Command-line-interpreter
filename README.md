# Command Line Interpreter



A Unix-inspired command line interpreter built from scratch in C++.



## Features

- Custom command set: echo, wc, head, tr, touch, rm, truncate, date, time, prompt, batch

- Full input/output redirection (< and >)

- Pipeline chaining between commands (|)

- Batch processing from text files

- Robust lexical, syntactic and semantic error reporting

- Test input files included (commands.txt, input.txt, hello.txt)



## Technologies

- C++

- Object-oriented design (class hierarchies, encapsulation, polymorphism)



## How to run

Open in Visual Studio, build the project and run. You can test commands interactively or feed `commands.txt` as a batch input.



## About

Built as a university project for the Object-Oriented Programming 1 course at the School of Electrical Engineering, University of Belgrade.
